第一层 外部上下文
USER/
用户品味语料
让 Claudio 真正属于你的那几个文件

BRAIN/
Claude Code
子进程调用 · 用 Max 订阅无偿 API key
claude -p --output json
MUSIC
NeteaseCloudMusicApi
歌曲检索 · 直链 · 歌词 · 推荐
search song_url lyric recommend
VOICE · I/O
Fish · Feishu · Weather · UPnP
声音 · 日程 · 天气 · 客厅功放
fish tts lark weather Naim
↓
第二层 本地大脑
ROUTER.JS
意图分流
简单指令直连 · 音乐走 ncm · 自然语言走 claude
CONTEXT.JS
提示词组装
taste + routines + 环境 + 历史 → system prompt
CLAUDE.JS
大脑适配器
spawn 子进程 · 解析 {say, play[], reason, segue}
SCHEDULER.JS
节律调度
07:00 规划 · 09:00 早间 · 小时情绪检查 · 日历 hook
TTS.JS
声音管线
Fish Audio → cache/tts/*.mp3 → /tts/<hash>.mp3
STATE.DB
状态 · 记忆
messages · plays · plan · prefs · 跨重启持久
↓
第三层 运行时聚合
CONTEXT WINDOW 组装盒子 · 每次触发按这 6 片粘成 prompt
① 系统提示词
prompts/dj-persona.md
② 用户语料
user/*.md
③ 环境注入
weather · calendar · now
④ 已检索记忆
state.db · plays
⑤ 用户输入 / 工具结果
/api/chat · ncm search
⑥ 执行轨迹
scheduler · webhook
MODEL · 前向过程 compute(fragments) → {say, play[], reason, segue}
→ ncm 解析 queue · tts 合成 say · WS 推 now-playing
↓
