## 第二部分：视觉风格建模 (Visual Spec)

### 1. 核心配色与质感

* **全局底色：** `#0d0d0d` (极黑)，但在视频中带有非常细微的噪点（Grain）效果。
* **播放器卡片（Light）：** 背景色为 `#ffffff`，圆角 `border-radius: 24px`。
* **文本高亮色：** 浅绿色背景 `#d1fadf`，文字颜色为纯黑。
* **点阵时钟：** 亮白色 `#ffffff`，配合强烈的 `text-shadow: 0 0 10px rgba(255,255,255,0.8)` 营造发光感。
* **毛玻璃效果：** 背景模糊 `backdrop-filter: blur(20px) saturate(180%)`。

### 2. 动效逻辑与时间轴

* **卡片切入动画：** * **缓动函数：** `cubic-bezier(0.22, 1, 0.36, 1)` (典型的 Out-Expo，开始极快，最后丝滑停住)。
* **时长：** `600ms`。


* **歌词高亮动画（Karaoke Style）：** * **机制：** 不是简单的逐字变色，而是类似一个绿色的“滑动滑块”覆盖在文字下方。
* **触发：** 由音频的时间戳同步触发。


* **波形图动画：** * **逻辑：** 并不是随机跳动，而是实时抓取频率数据。
* **参数：** 采样率 `fftSize: 256`，平滑系数 `smoothingTimeConstant: 0.8`。

- 底部的进度条，实现真正的镜像对称：

普通的“波浪进度条”是不行的了，而没有捕捉到视频中那种专业音频剪辑软件（如 Logic Pro 或 Audacity）特有的“**对称镜像波形**”质感。

视频中的底部进度条核心特征是：

1. **轴对称**：波形是以中心水平线为轴，向上下两端等距离延伸的。
2. **静态波浪+动态位移**：它看起来像是一段固定长度的音频采样，在播放时整段波形会有轻微的整体律动，而不是像正弦波那样飘。
3. **极高密度**：线条极细且极密，几乎没有缝隙。
4. **色彩截断**：进度点左侧是坚决的纯黑，右侧是淡灰。

1. **真正的镜像对称**：
* 我在 `.waveform-container` 中使用了 `align-items: center`。
* 在音频渲染逻辑中，高度不需要分“上半部分”和“下半部分”，当你给一个居中容器里的子元素设置高度时，它会**自动向上下两个方向等距延伸**，这完美还原了视频中音频采样线的质感。


2. **音频采样算法（Sampling）**：
* 之前的正弦波太像数学课件。这个版本使用了 `Math.sin(i * 0.1) * 15 + Math.sin(i * 0.5) * 10` 的叠加算法。
* 这样生成的波形会有明显的“波群”感，有些地方高，有些地方低，像一段真实的音频剪辑。


3. **色彩的“硬截断”**：
* 我放弃了渐变，使用了 `played` 类名进行 instant 切换。这保证了在任何进度下，左侧都是深沉的纯黑，右侧是优雅的浅灰，与白色背景形成对比。


4. **呼吸感 Jitter**：
* 我加了一个极其微小的 `jitter` 变量（正负 2px 内）。即便音乐在暂停或缓慢播放，波形线也会有细微的颤动，赋予 UI 一种“活着”的感觉。

- 顶部波形（Jagged Top）：引入了高频噪声算法，让波峰不再是圆润的弧线，而是锐利、错落、具有工业感的“碎裂”波形。
